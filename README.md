# Gym Management Website

A modern, responsive gym management system built with React, TypeScript, and Tailwind CSS.

## Features

- **Dashboard**: Overview of gym statistics and recent activities
- **Members Management**: Add, edit, and manage gym members
- **Classes Management**: Schedule and manage fitness classes
- **Trainers Management**: Manage trainer profiles and schedules
- **Equipment Management**: Track equipment status and maintenance

## Tech Stack

- React 18 with TypeScript
- Tailwind CSS for styling
- React Router for navigation
- Lucide React for icons
- Vite for build tooling

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
